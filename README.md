
#  Cloud-Based Health Record System

A secure, scalable, cloud-based web application for managing healthcare records using **Django** and **Firebase**.  
The system enables **doctors** to manage patient records and schedule appointments, while **patients** can securely view their medical history in real time.

---

##  Project Overview

Traditional healthcare record systems are often offline, manually maintained, and prone to data loss and accessibility issues.  
This project demonstrates how **cloud computing** can be used to modernize healthcare data management by providing:

- Centralized cloud storage
- Role-based access control
- Real-time data synchronization
- Secure authentication

---

##  Key Features

###  Doctor
- Secure login and registration
- Add new patient medical records
- Update existing records
- Schedule patient appointments
- View all patient data

###  Patient
- Secure login using Email or UID
- View personal medical records (read-only)
- View scheduled appointments
- Real-time updates from the cloud
